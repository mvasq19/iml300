Website Critique

Typography

1. Good Typography

I think this blog uses typography in a way that doesn't over power the content of the site. The choices the author/creator of the blog work well to create a pleasing design aesthetic.

2. Bad Typography

The type in this site seems to serve its only purpose with its functionality but it really isnt pleasing to the eye. The monochromatic color schemes chosen for the font (the ugly bright blue) does not make the site any better.
